Privacy Policy

Privacy Policy

Effective Date: November 24, 2025

At AxelBase Regex Explanation Generator, your privacy is not just respected — it is guaranteed by design. This tool was built from the ground up to be 100% client-side, meaning we collect absolutely no data from you.

No Data Collection — Ever

We do not collect, store, transmit, or log:

  • Any regex patterns you enter
  • Your IP address, location, or device information
  • Browser type, operating system, or screen size
  • Timestamps, session duration, or usage behavior
  • Cookies, localStorage (except temporary session state), or any tracking pixels

Your regex never leaves your browser. Even when you share a pattern via URL, it is Base64-encoded locally and decoded only in the recipient’s browser — no server is involved.

How the Tool Works

All parsing, explanation generation, catastrophic backtracking detection, and flavor comparison happen entirely in your browser using pure TypeScript and Web Workers. There is no backend, no database, and no analytics.

No Third-Party Services

We do not use:

  • Google Analytics, Plausible, Umami, or any analytics provider
  • Advertising networks or affiliate tracking
  • External fonts that phone home (all assets are self-hosted)
  • CDNs with tracking capabilities

GitHub Pages Hosting

This site is statically hosted on GitHub Pages. GitHub may log basic server access for security purposes (e.g., rate limiting), but AxelBase has no access to these logs and does not configure or request them.

Your Rights

Because we collect nothing, you have nothing to request deletion of. You are in full control at all times.

Children’s Privacy

This tool is safe and appropriate for users of all ages. No personal information is ever requested or stored.

Changes to This Policy

We may update this policy if new features are added. Any changes will be clearly dated above.

Contact

Questions? Open an issue on our GitHub repository.

Privacy isn’t a feature — it’s the foundation. Use this tool with complete confidence.
